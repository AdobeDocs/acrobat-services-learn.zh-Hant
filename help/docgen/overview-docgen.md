# 檔產生API教學課程

![檔產生API橫幅](../assets/docgenhero.jpg)

「檔產生」API從 Word 範本和 JSON 資料建立 PDF 和 Word 檔。

>[!NOTE]
>
>PDF 服務API包含「檔產生API」。

## 建立範本

產生API檔會接受檔範本 （包含範本標籤） 以及輸入資料，以產生最終檔。 系統會根據對應至資料輸入的實際值，將檔範本中的所有範本標籤替換為動態內容，藉此產生最終檔。
