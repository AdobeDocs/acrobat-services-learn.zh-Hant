# [!DNL Adobe Acrobat Services]個API教程

[!DNL Adobe Acrobat Services]有六個主API:

* [!DNL Adobe PDF Services API]
* [!DNL Adobe PDF Embed API]
* [!DNL Adobe Document Generation API]
* [!DNL Adobe PDF Electronic Seal API]
* [!DNL Adobe PDF Extract API]
* [!DNL Adobe PDF Accessibility Auto-Tag API]

後兩個API及其SDK作為付費產品的一部分捆綁到[!DNL Adobe PDF Services API]中。 [!DNL PDF Embed API]是免費產品。 這些API通過一組基於雲的現代Web服務自動生成、操作和轉換文檔內容。 它們可幫助您提供更簡單、更快速和品牌化的體驗，以便您能夠控制用戶與文檔的交互、簡化PDF工作流並促進使用和保留。 這些教程可幫助您快速使用[!DNL Adobe Acrobat Services]個API提供更簡單、更快速的品牌體驗。
