# Adobe PDF輔助功能自動標籤API教程

該AI支援的API自動標籤表、段落、清單、標題等，以利用輔助技術改進本地或掃描PDF的閱讀體驗。 它還標識讀取順序，以確保跨頁具有多個列或元素的資訊邏輯流。 將此應用於文檔積壓工作或新文檔工作流，並更輕鬆地實現法規遵從性。
