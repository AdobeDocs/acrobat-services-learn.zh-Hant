# Adobe PDF輔助功能自動標記API教學課程

此 AI 支援的API自動標記表格、段落、清單、標題等，藉助輔助技術改善原生或掃描的 PDF 閱讀體驗。 它也會識別閱讀順序，以確保在頁面之間使用多個欄或元素的邏輯資訊流向。 將此套用至文件備份或新檔工作流程，並更輕鬆地走向合規性。
