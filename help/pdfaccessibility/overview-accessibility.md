# Adobe PDF 無障礙自動標籤 API 教學

這個由 AI 驅動的 API 能自動標註表格、段落、清單、標題等，提升原生或掃描 PDF 的閱讀體驗，並結合輔助技術。 它也會標示閱讀順序，以確保資訊在多欄或多元素間的邏輯流動。 將此策略應用於文件積壓或新文件工作流程，並更輕鬆地邁向合規。
